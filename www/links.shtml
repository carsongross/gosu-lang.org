Links

Here are a variety of links related to Gosu. Have another link to suggest? Let us know.
  • DCEVM project site
    The Dynamic Code Evolution Virtual Machine (DCE VM) is a modified version of the Java HotSpot Virtual Machine (VM). The DCE VM supports any redefinition of loaded classes at runtime. You can add and remove fields and methods and make changes to the super types of a class using the DCE VM. The DCE VM is an improvement to the HotSpot VM, which only supports updates to method bodies. We recommend the use of the DCE VM for development against a running server.
  • Guidewire Development Blog
    Guidewire's platform developers have been blogging about the creation of not only the language, but other portions of our platform, as well as company life. It's a good place to learn a little about the internals of Gosu, or see some of the historical thinking that went into the evolution of Gosu.
  • Ronin Web Framework
    A very cool little Web UI Framework that uses some of the unique features of Gosu's typeloaders to let you put together a website really quickly.
  • Guidewire Software
    Guidewire is where Gosu started. See the History page for details of the orgins of Gosu. Guidewire graciously sponsors this website.
  • Lib Gosu
    Lib Gosu is an open source Ruby 2d game development library. We share a name, and coolness, but we're very different things
  • Lazy Gosuer
    The Lazy Gosuer is a compilation of snippets and examples. We think it demonstrates some of the features of the language pretty well, and is more fun than walking on hot coals.
