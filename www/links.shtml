Typeloaders and Projects that use Gosu

  • Ronin Web Framework
    A MVC based Web UI Framework that uses some of the unique features of Gosu's typeloaders to let you put together a website quickly. It uses a MVC paradigm, and convention over configuration, much like Rails.
  • Goson JSON typeloader
    A typeloader for Gosu that creates first-class objects from a JSON Schema.
  • Tosa
    An ORM framework for the Gosu language

Other stuff

  • DCEVM project site
    The Dynamic Code Evolution Virtual Machine (DCE VM) is a modified version of the Java HotSpot Virtual Machine (VM). The DCE VM supports any redefinition of loaded classes at runtime. You can add and remove fields and methods and make changes to the super types of a class using the DCE VM. The DCE VM is an improvement to the HotSpot VM, which only supports updates to method bodies. We recommend the use of the DCE VM for development against a running server.
  • Guidewire Software
    Guidewire is where Gosu started. See the History page for details of the orgins of Gosu. Guidewire graciously sponsors this website.
