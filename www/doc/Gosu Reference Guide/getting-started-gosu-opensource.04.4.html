The IntelliJ IDEA Plugin
Gosu includes an early version of a plugin for editing and debugging Gosu in the IntelliJ IDEA IDE. This early version of the IntelliJ IDEA editor plugin does not support the same level of functionality as the Eclipse plugin. For example, it supports the IDEA debugger and syntax coloring but does not support code completion using the Gosu type system. If you use IntelliJ IDEA, try the Gosu plugin and provide the Gosu team with feedback on it. For more information about providing feedback to the Gosu team, see "Get Ready for Gosu"
You can get the Gosu plugin directly from within the IntelliJ IDEA application:
1.
Launch IntelliJ IDEA.
2.
Click the menu item File â†’ Settings.
3.
In the left pane of the Settings window, click the item called Plugins.

5.

6.
In the list of plugins, find the row that says Gosu.
7.
Right click on that row and select Download and Install.
